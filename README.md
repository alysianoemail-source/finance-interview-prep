# 金融面试准备 Skill

帮你系统备战金融岗位面试的 Claude Skill。

面试准备最容易散的两件事：不知道这个岗位真正考什么，以及同时投几家时叙事搞混。这个 Skill 把这两个问题想清楚了。

---

## 怎么想的

**先搞懂岗位**，不只是读 JD。这个岗位日常做什么、产出什么、KPI 怎么考、2年后能去哪——还会自动搜面经和公司信息，官方渠道和小红书/脉脉一线信息都看。

**多个岗位分三类准备**，不重复劳动也不混淆：
- A 类共通的（行业认知、通用行为题）统一做一遍
- B 类每个岗位特有的（专业知识、特色案例）分开准备
- C 类叙事角度要调整——同一段经历，研究岗强调独立判断，客户岗强调推动结果，交易岗强调数据驱动。这点大多数人忽略，但考官很敏感

**个人匹配**：简历逐条改、知识按薄弱点补、面试题按重要程度排优先级。
