# finance-interview-prep

金融类岗位面试全流程准备 Skill，适用于 Claude。

支持单岗位和多岗位并行准备，覆盖从岗位研究到模拟作答的完整流程。

---

## 适用场景

- 投行、资管、券商、商业银行、金融科技等金融类岗位
- 同时准备 2-3 个不同岗位
- 3-4 天集中备战

---

## 文件结构

```
finance-interview-prep/
├── SKILL.md                        # 主流程（Claude 读取此文件）
└── references/
    ├── star-templates.md           # STAR 故事库模板（六维度）
    ├── narrative-types.md          # 差异化叙事类型（研究/客户/策略/管理）
    ├── question-bank.md            # 金融面试题库（动机/行为/技术/时事）
    └── knowledge-general.md        # 通用金融基础知识速查
```

---

## 安装使用

1. 下载 `finance-interview-prep/` 整个文件夹，放到你的 Claude Skills 目录
2. 在 Claude 对话中直接说：

```
帮我准备金融面试
```

或

```
我有三个金融岗位的面试，帮我准备
```

Claude 会自动触发此 Skill，引导你完成材料收集 → 自我诊断 → 岗位分析 → 准备规划 → 模拟作答的完整流程。

---

## 核心流程

| 步骤 | 内容 |
|------|------|
| [0] 材料收集 | 简历 + JD（最小启动），公司名 + 面试时间 |
| [0.5] 自我诊断 | 5个问题 → 个人化优先级地图 |
| [1] 岗位深度分析 | 工作内容/KPI/发展路径 + 面经/公司研究 |
| [2] 多岗位规划 | A类共通 / B类特异 / C类差异化叙事 |
| [3] 差异化叙事 | Why金融/Why公司/Why岗位，每岗位不同角度 |
| [4] 简历+案例库 | 逐条反馈 + STAR故事跨岗位标注 |
| [5] 知识补课 | fetch 官方来源，按薄弱点补 |
| [6] 问题预测 | P1/P2/P3 分级，反查漏洞 |
| [7] 模拟作答 | 按岗位分轮，逐题反馈 |
