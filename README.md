# 金融面试准备 Skill

一个帮你系统准备金融类岗位面试的 Claude Skill。

起因是觉得面试准备容易散——不知道从哪开始、准备了很多但不知道够不够、同时投好几个岗位容易混淆。这个 Skill 把整个流程串起来，每一步告诉你做什么、怎么做。

---

## 能帮你做什么

- 拆解岗位 JD，搞清楚这个职位真正要什么
- 自动搜面经（牛客、知乎、小红书），帮你判断哪些有参考价值
- 同时准备多个岗位时，分清楚哪些可以共用、哪些要分开准备
- 帮你打磨简历每一条，逐句给出修改建议
- 预测面试题，按重要程度排好优先级，不让你什么都背
- 最后做模拟面试，一题一题来，每题给反馈

---

## 适合谁

投行、资管、券商研究、商业银行、金融科技……只要是金融类岗位都可以用。

同时投了好几家、时间只有 3-4 天的情况下尤其有用。

---

## 怎么用

把这个文件夹放到你的 Claude Skills 目录，然后直接跟 Claude 说：

```
帮我准备金融面试
```

Claude 会先问你要简历和 JD，然后一步步带你走。
