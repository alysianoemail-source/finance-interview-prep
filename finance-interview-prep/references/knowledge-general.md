# 通用金融基础知识速查

## 财务报表三张表

### 利润表（P&L）
- 营收 → 毛利（毛利率）→ EBITDA → EBIT → 净利润
- 关键指标：毛利率、净利率、营收增速

### 资产负债表（Balance Sheet）
- 资产 = 负债 + 股东权益
- 关键指标：资产负债率、流动比率、速动比率

### 现金流量表（Cash Flow）
- 经营/投资/融资三类活动
- 自由现金流 = 经营现金流 - 资本支出
- **FCF 是最难造假的指标**

### 三表联动逻辑
- 净利润 → 现金流量表（调整非现金项目）
- 净利润留存 → 资产负债表所有者权益变化
- 资产负债表变化 → 现金流量表营运资本变动

## 常见估值方法

### DCF（现金流折现）
- 核心逻辑：资产价值 = 未来现金流的现值之和
- 关键假设：折现率（WACC）、终值增长率
- 优点：理论最严谨；缺点：对假设极敏感

### 可比公司法（Comps）
- 用同行业类似公司的估值倍数来估值
- 常用倍数：EV/EBITDA、P/E、P/B、P/S
- 注意：需要调整差异（规模、增速、盈利能力）

### 可比交易法（Precedent Transactions）
- 用历史并购交易价格来估值
- 通常包含控制权溢价（20%-30%）
- 适用：并购定价参考

### 资产法
- 适用：重资产行业、清算估值
- 调整账面价值至市场价值

## 金融产品基础

### 固定收益
- 债券价格与利率反向运动
- 久期：利率变动 1% 对应价格变动幅度
- 信用利差：高于无风险利率的额外回报

### 权益
- Beta：相对市场的波动性
- Alpha：超额回报
- 股息率 vs 资本利得

### 衍生品
- 期权：买权（Call）/ 卖权（Put）
- 期货：锁定未来买卖价格
- 互换（Swap）：利率互换、货币互换

## 宏观经济关键指标

| 指标 | 关注点 |
|------|--------|
| GDP 增速 | 经济总量变化 |
| CPI/PPI | 通胀压力 |
| 利率（央行政策利率） | 资金成本基准 |
| 汇率 | 跨境资本流动 |
| PMI | 制造业景气度领先指标 |
| 失业率 | 劳动力市场紧张程度 |

### 利率与资产价格关系
- 利率上升 → 债券价格下跌，股票估值承压，房地产降温
- 利率下降 → 相反方向

## 面试常考概念

**WACC（加权平均资本成本）**
= 债务成本 × (1-税率) × 债务权重 + 股权成本 × 股权权重

**CAPM（资本资产定价模型）**
股权成本 = 无风险利率 + Beta × 市场风险溢价

**EV（企业价值）**
EV = 市值 + 净债务（债务 - 现金）
> EV 是收购一家公司的真实成本

**杠杆收购（LBO）逻辑**
用大量债务收购企业，通过运营改善和债务偿还提升股权回报

**净营运资本（NWC）**
NWC = 流动资产 - 流动负债
NWC 增加 = 消耗现金（对现金流不利）
